# RangeImagePlanar
## Dependencies

The code is based on the following prerequisites:
- PCL
- GDAL
- Eigen3
- OpenCV

## Compilation

prerequisites: cmake version>=3.0

```
1. git clone https://github.com/AndrewAndJenny/RangeImagePlanar.git
2. cd RangeImagePlanar
```

**in the cmake-bash**
- Linux

```cmake
1. mkdir build && cd build
2. cmake ..
3. make
```
